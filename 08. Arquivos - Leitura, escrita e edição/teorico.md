# Manipulação de arquivos externos com Python

Os modos de leitura são:
- r: Read /Leitura - Padrão
- w: Write / Escrita - Sobreescreve o arquivo original
- a: Append / Acréscimo - Não sobreescreve o arquivo original

## Write

O modo write pode escrever em um arquivo, se o arquivo não existir ele será criado.

Quando usado o modo write, o Python sempre irá apagar o arquivo original e criar um novo no lugar.

## Append

O modo write adiciona novos dados no final de um arquivo já existente.
