# Escopo

No Python, escopo é o que define o ciclo de vida de uma variável.

Toda variável criada pelo Python aloca um espaço na memória RAM do dispositivo. Esse espaço fica alocado até que a aplicação seja finalizada.

No entanto, as variáveis presentes dentro de funções tem um comportamento diferente, existindo apenas enquanto a função é executada e em seguida são automaticamente desalocadas da memória.
